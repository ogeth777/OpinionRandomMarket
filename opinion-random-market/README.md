# Polymarket Random Roulette

A web app to randomly explore active Polymarket markets.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack
- React + Vite
- Tailwind CSS v4
- Framer Motion
- Polymarket Gamma API
